# Votest App (Web)

## Setup on local & development

- Node.js v8 via [nvm](https://github.com/creationix/nvm)
  - `$ nvm use v8`
- update `.env` (using `.env.sample` sample)
  - set `NODE_ENV=development`
- `yarn test`
- `yarn start:dev`

## Setup on production

- `yarn test`
- `yarn start`
